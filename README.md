# Time-Date-Widget-JS
Стильный и удобный виджет на JS+CSS, использует Picture-in-Picture окно в браузерах Chrome, Chromium, Opera, Yandex

Фичи проекта:

- **Zero Install & Универсальность:** Работает на любом устройстве через браузер, вам не нужно устанавливать сторонние приложения! Вы просто захотели дополнить свой рабочий стол виджетом, перешли по ссылке и нажали на кнопку "Окно" - и виджет у вас на экране поверх других окон! Код легко встраивается в другие сайты, Notion, OBS Studio, Wallpaper Engine или Rainmeter.
- **Document PiP API:** Перенос полноценного HTML/CSS/JS контента в отдельное плавающее окно поверх всех остальных приложений.
- **Динамический контекст:** Автоматическая перепривязка логики обновлений времени при перемещении виджета между окнами.
- **Pixel-Perfect UI:** Глубокие градиенты, полупрозрачные элементы и размытые тени для максимального улучшения пользовательского опыта с точки зрения эстетики, современнго дизайна и удобства.

Технологический стек:

- **HTML5** (Семантическая разметка, изоляция структуры виджета)
- **CSS3** (Custom Properties, Flexbox, CSS Filters, Box-shadows)
- **Vanilla JavaScript** (Modern ECMAScript, Document Picture-in-Picture API, DOM Manipulation)

Известные ограничения:
- **Неработоспособность PiP-оконного режима в Firefox**: В браузере Mozilla Firefox функционал PiP ограничен до простого видеоплеера для фонового просмотра медиа. В отличии от Chromium в нём не поддерживается выполнение полноценного кода html, js и css.
- **Не скрываемый системный заголовок окна от Chromium**: Ограничение архитектуры безопасности API, не позволяющее скрывать заголовок окна. Немного мешает задуманному мной дизайну, но является вынужденной мерой безопасности для защиты от фишинга.
